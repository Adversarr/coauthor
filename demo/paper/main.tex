\begin{abstract}
This document describes Seed, a task-driven and event-sourced assistant-team runtime for local workspace execution. Seed treats the user's goal as the seed for planning, tool use, delegation, and review. The system emphasizes auditability, controlled risky actions, and reusable domain-agnostic workflows.
\end{abstract}
